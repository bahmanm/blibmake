# `bmakelib.MIN_MAKE_VERSION`

Minimum Make version supported.
Anything older either breaks bmakelib or can cause it to behave in unexpected ways.

---

# `bmakelib.newline`

Expands to a single newline (\n)

---

# `bmakelib.comma`

Expands to a comma (,)

---

# `bmakelib.backslash`

Expands to a backslash (\)
